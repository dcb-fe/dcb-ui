# Notify

::: tip 说明

下述演示为了视觉效果将 `renderInPlace` 属性设为了 `true` 以让弹出层原地渲染，实际使用最好不要添加该属性以让通知渲染到 `document.body` 下避免样式污染问题。

但如果你要自定义样式，譬如自定义通知背景色、字体颜色等，可将 `renderInPlace` 设为 `true`。

:::
