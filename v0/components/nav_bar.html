# # API

# # 属性

名称 说明 类型 默认值
title
导航标题 string
leftText
左侧文案 string
rightText
右侧文案 string
search
是否展示搜索 boolean false
placeholder
搜索默认提示文案 string 搜索关键词
share
是否展示分享 boolean false
close
是否展示关闭 boolean false
background
导航背景色 string
searchBgColor
搜索框背景色 string rgba(245, 245, 245, 1)
fixed
是否浮动布局 boolean false
safeTop
刘海屏安全区，非多彩宝环境设置为false boolean true
