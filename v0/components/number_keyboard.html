# # API

# # 属性

名称 说明 类型 默认值
value
v-model
当前输入值 string
hideOnClickOutside
点击外部时是否收起键盘 boolean true
show
是否显示键盘 boolean false
extraKey
额外按键的内容 string | array
theme
样式风格，可选值为 custom string def
title
键盘标题,空则不展示 string
closeButtonText
关闭按钮文字，空则不展示 string
randomKey
是否将通过随机顺序展示按键 boolean false

# # 插槽

无

# # 事件

名称 说明 参数
close 点击关闭按钮时触发 无
delete 点击删除键时触发 无
blur 点击关闭按钮或非键盘区域时触发 无
select 点击按键时触发
名称 说明 类型
key key: 按键内容 string
