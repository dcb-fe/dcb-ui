# # API

# # 属性

名称 说明 类型 默认值
content
徽标内容 string | number
color
徽标背景颜色 string #ee0a24
dot
是否展示为小红点 boolean false
max
最大值，超过最大值会显示 {max}+，仅当 content 为数字时有效 string | number 99

# # 插槽

名称 说明 参数
default 徽标包裹的子元素 无
content 自定义徽标内容 无

# # 事件

无
